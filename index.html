Who We Are

PyLadies Kaohsiung - 高雄為基地的女性技術社群

PyLadies Kaohsiung 是 PyLadies 國際組織的一員，以高雄為基地推動女性技術社群的發展， 致力於打造讓女性能安心、自信地學習、交流與成長的環境。
我們舉辦各種工作坊、經驗分享等交流活動，讓大家能透過 Python 提升技術、拓展人脈與視野， 進而在科技領域中成為主動的參與者與領導者。
同時也積極與在地及全台社群建立連結、相互支持，讓整體技術社群更加茁壯。

Our Values

What we value most

開放與共創

安心學習與交流的空間

我們希望打造一個讓女性能自在提問、安心表達的學習與交流環境。 在這裡，沒有比較與評價，只有彼此支持與共同成長。

共識形成的社群文化

PyLadies Kaohsiung 屬於每一位參與者。 每個意見都被重視，我們願意傾聽所有回饋，並與所有志工間討論、凝聚共識， 讓社群的方向能真正反映成員的聲音與價值。

透明與信任的基礎

作為非營利社群，我們希望能讓金流與資源使用的透明化。不僅如此，我們也重視參與者的溝通管道， 讓每位成員的意見都能被確實傳遞、被傾聽，並反映在社群運營中。

成長與傳承

學習 Python 邁向成長

不論你是初學者或已有經驗的開發者， 在這裡都能透過學習 Python 提升技術、建立信心與增加人脈， 與夥伴一起探索更廣闊的可能性。

陪伴與協助，勇敢分享知識

透過分享，每位女性都有機會影響更多人。我們提供內容諮詢與試教回饋，幫助女性講者建立自信。此外也利用長期合作的工作坊教材，提供完整的培訓制度，讓志工更容易站上舞台，不需自行規劃教材，更能專注在分享與成長。

傳承與可持續的社群精神

我們相信每個人都有成為領袖的潛力。 PyLadies Kaohsiung 正在建立更完善的制度，領導者也會主動分享經驗與舞台， 讓積極的志工都能獲得帶領與貢獻的機會。也會透過制度確保共治精神， 讓社群能持續維持開放與共同成長的文化。

F.A.Q

Frequently Asked Questions

怎麼參加 PyLadies Kaohsiung 的活動？

我們主辦的活動都會發到 KKTIX 上，或者追蹤我們粉絲專頁獲得最新活動消息。

活動是否會收費？

視活動舉辦的地點而定，如果有場地費，會是需要自行負擔的。目前在 PyLadies Kaohsiung 主辦的活動，積極爭取下，感謝有不少單位願意無償提供場地，因此許多線下活動場次可以讓女性得以免費參加，也呼籲成功報名的學員珍惜這些機會。

為什麼 PyLadies Kaohsiung 主辦的線下活動幾乎都僅限女性參加？

因為目前在資訊科技領域的女性比例仍然算低， 許多對 Python 有興趣，或想在技術上更進一步的女性，在以男性為多數的環境中學習，常會感到壓力或不自在；就像男性走進幾乎全是女性的瑜伽等律動活動一樣，難免也會感覺緊張。
而 PyLadies Kaohsiung 的核心目標是想打造一個能讓女性安心自信地學習、討論溝通的環境， 讓更多喜歡 Python、想進入科技領域的女生，有機會成為主動的參與者與領導者。讓參與者(學員、志工、講師)有自在的空間一直是我們首要目標，也因此主辦的線下活動目前基本上僅開放女性參加。
我們真心希望，在不遠的將來，當資訊科技領域中的性別比例趨於一致、彼此都能互相尊重、人人都能自在參與時，這樣的性別限定活動將不再必要，屆時我們也會以更開放的方式歡迎所有喜愛 Python 的朋友，一起交流、一起成長～

男性有機會參加 PyLadies Kaohsiung 的活動嗎？

➡︎ 講師：沒有任何限制，歡迎認同我們理念的朋友報名擔任講師。
➡︎ 志工：招募以女性為主要對象。視活動性質與需求，可能也會開放不限性別的夥伴一同協力，期待更多人共同推動友善的學習環境。
➡︎ 學員：線下活動如果是 PyLadies Kaohsiung 主辦的目前基本上僅限女性參加；若為與其他單位合辦或受邀參與的場次，則依該主辦單位的報名規則而定，是有機會可以參加的。

PyLadies Kaohsiung 只會在高雄辦線下活動嗎？

雖然 PyLadies Kaohsiung 專注在高雄，但有機會都會到全台各地舉辦活動喔～

我可以在 PyLadies Kaohsiung 分享我的 Python 經驗嗎？

當然沒有問題，請透過下方的各種社群平台聯絡我們。

如何成為 PyLadies Kaohsiung 的志工？

我們會在粉絲專頁上發佈任何志工招募訊息，請隨時關注～

台灣好像還有其他 PyLadies 的社群？你們的關係是？

PyLadies 是一個國際的組織，每個地區都可以舉辦自己的 PyLadies 社群，同樣都是以推廣 Python 與支持女性在科技領域成為參與者與領導者為目標，但彼此之間獨立運作。各地社群會依照在地的特性舉辦活動，也會不定期合作或互相支援。
